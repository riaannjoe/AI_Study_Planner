# AI Study Planner 📚🤖

A smart AI-powered study planner that helps students create personalized study schedules based on subjects, available time, and exam deadlines.

Built using Python, Flask, and AI-based planning logic to make studying more organized and productive.

---

## ✨ Features

* 📅 Generate personalized study plans
* ⏳ Set available study hours per day
* 📚 Add multiple subjects
* 🎯 Plan based on days left before exams
* ⚡ Simple and beginner-friendly interface
* 🤖 AI-inspired study distribution logic
* 🌐 Flask-based web application

---

## 🛠️ Tech Stack

### Frontend

* HTML
* CSS

### Backend

* Python
* Flask

### Logic

* Custom AI-based scheduling algorithm

---

## 💡 How It Works

1. Enter your subjects
2. Add number of days left
3. Enter study hours per day
4. The planner intelligently distributes study sessions
5. Get your personalized study schedule instantly
